# 错误码

本文档列举调用物联网平台API出错时，返回的错误信息。入参数据格式错误、超出限定值、入参缺少必需参数等错误修改，请参见具体API文档的请求参数描述。

## 系统错误码

以`iot.system`开头的错误码为系统相关错误码。

|错误码|描述|
|:--|:-|
|iot.system.SystemException|系统异常。 请稍后重试。 |
